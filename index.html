July, 2023

FIREDpy

The FIRe Event Delineation tool created by Earth Lab at CU. I was briefly apart of some summer work that fixed some edge-case issues on the program. It's a Python based CLI that classifies individual burn detections into discrete events from 2001 to near-present. It returns both a table and shapefiles of fire events for a time and place (Balch et. al., 2020).
